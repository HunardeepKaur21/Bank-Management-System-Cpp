# Bank Management System (C++)

A console-based Bank Management System built using C++.

## Features
- Create Account
- Display All Accounts
- Deposit Money
- Withdraw Money
- Loan Facility
- Search Account

## Technologies Used
- C++
- File Handling (fstream)

## How It Works
- Account data is stored in a file (accounts.txt)
- Operations update file using temporary file logic

## Learning Outcomes
- Object-Oriented Programming (OOP)
- File Handling
- Real-world logic implementation
